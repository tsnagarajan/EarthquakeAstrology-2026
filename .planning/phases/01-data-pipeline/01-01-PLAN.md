<objective>
Download all M5.5+ earthquake records from the USGS FDSNWS Event API for the period 1900–2026, paginating by decade to avoid the 20,000-record API limit, and save as a single CSV to disk.

Purpose: This raw catalog is the earthquake ground-truth that will become the EQIndicator target variable in Phase 2. If any decade is silently truncated, the target label distribution is corrupted and model evaluation is meaningless.
Output: data/raw/usgs_earthquakes.csv (~50,000–150,000 rows), pyproject.toml with Python dependencies
</objective>

    - Script exits with code 0 on success; exits non-zero on network failure after retries
  </behavior>
  <action>
    Create `pipeline/data/usgs.py` as a standalone executable script with the following structure:

    **Imports:** requests, pandas, time, tqdm, pathlib, sys, logging, argparse, datetime, dotenv

    **Constants:**
    ```python
    USGS_URL = "https://earthquake.usgs.gov/fdsnws/event/1/query"
    MIN_MAG = 5.5
    OUTPUT_PATH = Path("data/raw/usgs_earthquakes.csv")
    TRUNCATION_LIMIT = 20_000
    ```

    **`fetch_decade(start_year, end_year)` function:**
    - Builds params dict: format="csv", minmagnitude=5.5, starttime=f"{start_year}-01-01", endtime=f"{end_year}-12-31", orderby="time-asc"
    - Makes GET request with requests.get, timeout=60, retries up to 3 with exponential backoff (2s, 4s, 8s)
    - Raises RuntimeError if response.status_code != 200
    - Parses response text via pd.read_csv(io.StringIO(response.text))
    - CRITICAL CHECK: if len(df) == TRUNCATION_LIMIT, log a WARNING and raise RuntimeError("Decade {start_year}-{end_year} hit 20k limit — sub-chunk required")
    - Returns DataFrame

    **`fetch_all(start_year, end_year, chunk_years)` function:**
    - Iterates year ranges in chunk_years steps (default: 5 years to stay safely under 20k)
    - Uses tqdm progress bar over chunks
    - Calls fetch_decade for each chunk
    - Sleeps 0.5s between requests to be respectful of the API
    - Concatenates all chunk DataFrames
    - Deduplicates on 'id' column (keep first) in case chunk boundaries overlap
    - Returns merged DataFrame

    **`validate_result(df)` function:**
    - Asserts required columns exist: ['time', 'latitude', 'longitude', 'depth', 'mag', 'place', 'type']
    - Asserts df['mag'].min() >= MIN_MAG
    - Asserts len(df) > 50_000, else logs a warning (not an error — catalog may be sparse pre-1950)
    - Returns True

    **`main()` entry point:**
    - argparse: --start-year (default 1900), --end-year (default current year), --chunk-years (default 5), --output (default OUTPUT_PATH)
    - Calls fetch_all, validate_result
    - Writes to output path via df.to_csv(output, index=False)
    - Prints summary: "Downloaded {len(df)} events. Saved to {output}"

    **Guard:** `if __name__ == '__main__': main()`

    Logging: Use logging.basicConfig(level=logging.INFO). All warnings about truncation or low counts go to stderr via logging.warning.
  </action>
