## Naming Conventions

**Files:**
- Notebook files: CamelCase with descriptive names (`Durga Copy of TSN ML Worksheet Full.ipynb`)
- CSV data files: snake_case with dates/versions (`2024_09_18 Astrology earthquake data processed.csv`)
- Excel predictions: YYYY format (`2019EQPredictions.xlsx`, `2020EQPredictions.xlsx`)
- Archive: Prefixed with eq* or similar (`eqclean10272019.csv`, `eqAstroTable*.csv`)

**Notebook Cells:**
- Markdown section headers: `## Section Name` (double hash for major sections)
- Code cell descriptions: Inline comments or preceding markdown cells
- Variable names in cells: camelCase for data operations (`eqCleanP1`, `linkList`, `eqAstroTableDf`)
- Function names: camelCase (`genLinkGivenInfo`, `createDfFromLink`, `planetAspectCalc`)

**Directories:**
- Project root: PascalCase (`Earthquake 2026`)
- Subdirectories: PascalCase (`Archive`, `Tropical Data`, `EQ papers`)

## Where to Add New Code

**New Feature Engineering Logic:**
- Primary code: Add functions or inline logic to `Durga Copy of TSN ML Worksheet Full.ipynb` cells 56-126
- Testing approach: Add new cell with Counter() or descriptive analytics (cells 127-139 pattern)
- Integration: Reference new feature columns in MLData selection (cell 143 logic)

**New ML Model:**
- Implementation: Add to cells 140-166 section (`## Machine Learning Supervised` or new section)
- Pattern: Import model from sklearn, fit on MLData/EQIndicator, evaluate via Counter() of predictions
- Comparison: Add side-by-side with existing LogisticRegression and K-Means

**Data Processing Function:**
- Location: Define function in appropriate layer cell range (e.g., cells 35-55 for scraping, cells 70-126 for feature engineering)
- Pattern: Take DataFrame input, return modified DataFrame with additional columns
- Example: `planetAspectCalc` in cell 70 calculates aspects as separate columns

**Data Import from New Source:**
- Implementation: Add pd.read_csv() or new data loading cell in cells 1-27
- Merge approach: Combine with existing earthquake dataframe via pd.concat() or merge()
- Label addition: Assign EQ_Happened flag before sending to feature pipeline

**Exploratory Analysis:**
- Location: New cells in "Sandbox - TSN Explore" section (cells 77-113) or new markdown section
- Pattern: Load data, apply Counter() or value_counts(), visualize with matplotlib
- No persistence required; output to notebook cell display
